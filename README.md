# AI4MedImaging

This is a simple project made with the front in react and the back in flask. The purpose of this project is a test for the company AI4MedImaging

# Docker Build and Run Instructions

## Build Steps

1. Ensure Docker is installed.
   Make sure you have Docker installed on your system. You can check if Docker is installed by running the following command:

```sh
docker --version
```

2. Navigate to the project's root directory.
   Open a terminal window and navigate to the directory that contains the project's Dockerfile and other source files.

3. Build the Docker image:

```sh
docker build -t image-processing-app .
```
